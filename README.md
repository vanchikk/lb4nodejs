**Системные требования**
Для запуска проекта необходимы:

Node.js (LTS версия) - **https://nodejs.org/en/download**

npm (устанавливается вместе с Node.js)

**Инструкция по запуску**
Скачай проект. Нажми на кнопку Code -> Download ZIP или склонируй репозиторий.

Распакуй архив и открой папку с проектом в терминале (или в VS Code).

Установи зависимости. В терминале введи команду:

**В терминале:**
npm install
Это создаст папку node_modules и загрузит необходимые библиотеки (Express, Socket.IO).
Если тебе нужно ещё раз запустить программу, используй npm start.

Запусти сервер. Выполни команду:

**В терминале**
node index.js
После этого в консоли должно появиться сообщение: listening on *:3000.

**Открой браузер по адресу:**
http://localhost:3000

Важно для теста: Чтобы увидеть магию чата, открой две разные вкладки по этому адресу. Напиши сообщение в одной — и оно мгновенно появится в другой без перезагрузки страницы.

Используемые технологии 
Express — обработка маршрутов и статических файлов.

Socket.IO — двусторонний обмен данными между клиентом и сервером.

jQuery — быстрая манипуляция элементами страницы на стороне клиента.
